# Meta AI Prompts — Sprites NPC Mundo 1 + 2

Estilo Meta AI já validado (Escriba Imhotep). Padrão:

- Full body, vertical 2:3, **sandálias/pés visíveis** (essencial: anchor pivot nos pés).
- Cartoon afro-cêntrico estilo Kirikou (Michel Ocelot), flat shading + 1 sombra.
- Pele escura africana autêntica.
- **No text, no title, no watermark** (Meta vai sempre adicionar watermark; remover em Photopea depois).
- **Plain colored background** (não cena complexa) — facilita remover fundo depois.
- Iterar 3-4 takes; escolher melhor.

Cole o prompt todo no chat Meta AI (`meta.ai`) e gera imagem.

## Pipeline pós-geração

Para cada PNG gerado pelo Meta AI:

1. **Photopea** (online grátis: `photopea.com`):
   - Open imagem JPEG.
   - Magic wand (W) → click fundo → Delete (Ctrl+Del).
   - Clone stamp watermark "Meta AI" canto.
   - Apagar título se houver.
   - Resize: Image → Image Size → 512×768 PNG-32.
2. **Export PNG**: File → Export → PNG-32 transparent.
3. **Drop** no path indicado (`kids/game/assets/world1-X/...` ou `world2-X/...`).
4. **Avisa-me** com o path quando dropares — eu adiciono `img:` ao NPC data + bump SW + precache.
