# Contribuir com o Sankofa

Obrigado pelo interesse em ajudar.

## Status atual: fechado para PRs externos

Sankofa está em **fase inicial de estabilização**. Para evitar que decisões de
arquitetura e direção pedagógica fiquem dispersas antes da base solidificar,
**Pull Requests externos não estão sendo aceitos por enquanto**.

O repositório é **público para leitura, auditoria e estudo** — sob licença MIT
para o código e CC BY-SA 4.0 para o conteúdo educativo. Forks técnicos para uso
próprio são livres pela própria licença, mas não serão integrados a esta base
neste momento.

## Quando reabrir

A política mudará quando:

- A estrutura do código estiver mais sedimentada (estimativa: depois do
  primeiro piloto pedagógico em escola).
- Houver um guia de contribuição com convenções claras
  (estilo de código, escrita de enigmas, validação histórica, testes).
- Houver capacidade de revisão constante de PRs sem prejudicar o ritmo
  do desenvolvimento principal.

A reabertura será anunciada aqui e na home do projeto.

## Issues e Pull Requests

- **Issues abertas externamente serão lidas mas não respondidas individualmente.**
  Migra a sugestão para o formulário em-app, que tem campo de tipo, screen,
  versão e contato opcional — formato mais útil para a equipe de desenvolvimento.
- **Pull Requests serão fechados sem merge** durante esta fase, com link para
  este documento. Não é rejeição do conteúdo — é foco operacional.

## Licença e atribuição

Quando a contribuição reabrir, todo PR aceito implica concordância com:

- **Código:** licença MIT.
- **Conteúdo educativo (enigmas, fragmentos, textos):** Creative Commons
  CC BY-SA 4.0.
- Conformidade com a licença CC BY-SA IGO 3.0 da UNESCO para qualquer
  conteúdo derivado da *General History of Africa* (atribuição obrigatória,
  disclaimer obrigatório, mesma família de licença, sem uso de logo UNESCO,
  sem uso de fotografias da publicação original).
