# Raw artefacts

Materiais de origem usados como base para o jogo. Mantidos em `docs/raw/` para
preservar histórico, mas **não são consumidos pelo build**. A acentuação aparece
quebrada em alguns deles porque vinham de OCR/colagens.

| Arquivo                     | O que é                                                  |
|-----------------------------|----------------------------------------------------------|
| `projeto-v1.0.md`           | Documento de projeto v1.0 completo (35 KB).              |
| `test-code-prototype.html`  | Protótipo HTML monolítico que inspirou o visual atual.   |
| `test-code-mock.txt`        | Mock ASCII da tela de perfil.                            |
| `code (*).txt` / `code.json`| Anotações iniciais e fragmentos brutos de conteúdo.      |

A versão **canônica e atualizada** do conceito está em `docs/CONCEITO.md`. O roadmap
em `docs/ROADMAP.md`. O motor de áudio em `docs/AUDIO.md`.
