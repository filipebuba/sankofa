# Campanha de Lançamento — Sankofa Universo

> Plano coordenado de 12 meses para lançar TODAS as superfícies do universo Sankofa em sequência amplificadora. Cada drop alimenta o seguinte.

---

## Tese da campanha

**O lançamento não é um momento — é um arco de 12 meses.** Cada formato (jogo, música, vídeo, podcast, livro) reaparece de tempos em tempos para reciclar atenção e funelar pra outro formato.

Princípios:
1. **Mês de pré-lançamento** silencioso construindo lista de espera
2. **Drop 1** explosivo: jogo + 1 música + 1 vídeo + 1 episódio podcast no mesmo dia
3. **Cadência mensal** com novo drop musical
4. **Eventos físicos** acoplados a momentos públicos (Dia da África, Consciência Negra)
5. **Crowdfunding** quando atenção pública estiver no pico

## MITIGAÇÃO DE RISCOS

| Risco                                          | Mitigação                                           |
|------------------------------------------------|-----------------------------------------------------|
| Lançamento mês 1 fraco (poucos jogadores)     | Tem 11 meses pra construir. Cada drop melhora.      |
| Acusação de "comercializar tema racial"        | Open source + 50% receita pra fundo afro            |
| Crowdfunding falha                             | Edital PNAB cobre se cair                           |
| Convidado podcast importante recusa            | Lista alternativa de 30+ nomes                      |
| Burnout                                         | Cronograma aceita atraso de 1 mês sem quebrar       |
| Imprensa ignora                                 | Microinfluencers compensam (50 colabs > 1 jornal)  |
| YouTube ban (algoritmo confunde tema)          | Mirror em PeerTube + Vimeo                          |
| Conflito UNESCO sobre uso da HGA               | Carta UNESCO Brasil em mãos antes do mês 1         |
| Vercel cai / domínio perdido                   | Backup Netlify + GitHub Pages                       |
