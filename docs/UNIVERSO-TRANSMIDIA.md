# Universo Transmídia Sankofa

> **Tese**: Sankofa é maior que um jogo. É um *universo narrativo afrocentrado* que se manifesta em jogo, música, vídeo, podcast, livro, oficina, série e tabuleiro. Todos os formatos compartilham os 8 mundos, as 9 Casas Reais, os personagens, a estética papiro/ouro/terra e a moeda **cauris**. Cada formato alimenta o outro.

---

## 1. Princípios do universo

| Princípio                    | Explicação                                                                            |
|------------------------------|---------------------------------------------------------------------------------------|
| **Continuidade narrativa**   | Mansa Musa do jogo é o mesmo do clipe musical, do podcast e do livro infantil       |
| **Lógica de jogo em tudo**   | Cada formato tem cauris, missões, casas reais, ranking — não só o jogo               |
| **Open by default**          | Música, vídeo, livro, áudio — todos sob CC BY-SA 4.0 (exceto onde imposto contratual)|
| **Estética unificada**       | Mesmas cores, mesma fonte, mesma identidade Sankofa em todas as superfícies          |
| **Cross-funneling**          | Cada vídeo do YouTube termina com convite a outro formato                            |
| **Adaptação por idade**      | Sankofa Kids (4-9), Sankofa (10-17), Sankofa Adultos (18+), Sankofa Sábios (50+)     |

## 3. SANKOFA SOM — divisão musical

### 3.1. Conceito do projeto musical

> **Cada um dos 8 mundos vira um EP de 4-6 faixas. Total 32-48 músicas em 18 meses. Estilo: afrobeat + samba + maracatu + griot mandinga + amapiano + rap consciente.**

**Casas Reais viram personas musicais**:
- **Mansa Musa** = MC mandinga com kora
- **Kandake Amanirenas** = cantora afronubiana com tambor falante
- **Faraó Hatshepsut** = cantora afrofuturista com sintetizador árabe
- **Asantehene** = sambista de roda + atabaque
- **Nzinga** = rapper bélica + percussão angolana
- **Dandara** = cantora MPB-quilombola
- **Steve Biko** = poeta spoken word + jazz sul-africano
- **Sundiata** = griô de djembé pesado

### 3.2. Estrutura de cada música

Aplica lógica de enigma:
1. **Estrofe 1** — pista poética (mesma do jogo)
2. **Pré-refrão** — segunda pista
3. **Refrão** — pergunta direta ("Quem sou eu?")
4. **Estrofe 2** — terceira pista + contexto
5. **Refrão**
6. **Ponte** — revelação parcial
7. **Outro** — nome do personagem cantado, "fragmento desbloqueado"

## 11. Lógica de jogo aplicada — sistema de cauris cross-mídia

> **O jogador acumula cauris em qualquer formato. Vê-los no jogo final.**

| Ação                                          | Cauris  |
|-----------------------------------------------|---------|
| Resolver enigma no jogo                       | 5-25    |
| Insere código secreto da música do mês       | 50      |
| Insere código do vídeo da semana             | 30      |
| Termina episódio do podcast (autoreport)      | 20      |
| Lê 1 livro Sankofa (declara via formulário)   | 100     |
| Comenta com tag #sankofa em rede             | 5/dia   |
| Compartilha selo real                         | 10      |
| Termina curso aberto                           | 200     |
| Comparece evento físico (QR check-in)         | 75      |
| Indica amigo (que cria perfil)                | 30      |
| Frequência semanal (toda semana ativa)        | 50      |

Tudo entra no mesmo placar. Liga semanal mistura jogadores que jogaram o jogo + ouviram podcast + viram clipe musical.

## 16. Por que isso funciona

4. **Cada formato funela o outro** — quem ouve a música procura o jogo; quem joga procura o podcast.
5. **Lei 10.639 é mercado cativo** — 137 mil escolas precisam de material.
6. **Tu já és multidisciplinar** (CS + Ciências Humanas) — o universo cabe em uma cabeça só.
