# Sankofa skills

Project-scoped skills for Claude Code, auto-loaded when invoked from this repo.

| Skill                  | Quando usar                                                        |
|------------------------|--------------------------------------------------------------------|
| `sankofa-add-enigma`   | Adicionar um novo enigma (validação, schema, fonte HGA).           |
| `sankofa-add-world`    | Destravar/adicionar mundo, ligar arte e conquistas.                |
| `sankofa-tune-audio`   | Ajustar instrumentos, eventos sonoros, ambient e timbre.           |

Os arquivos `SKILL.md` em cada subpasta são lidos pelo Claude Code quando o skill é
invocado. Para entender melhor cada um, abra a pasta correspondente.
